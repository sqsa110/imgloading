# 图片预加载插件

## 参数列表

** box **            //需要进行图片加载的区域  默认 'body'

** el **             //需要进行图片加载的元素  默认 'img'

** lazyload **       //是否需要懒加载    默认false为不需要

** adaptiveSizeW **   //从标签宽度自适应图片大小   默认false为不需要

** adaptiveSizeH **   //从标签高度自适应图片大小   默认false为不需要

** callback **        //回调函数  分别传入3个参数 图片url,图片宽度，图片高度
